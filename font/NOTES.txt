Font Types Supported in PDF
---------------------------

Type1 - glyph shapes using Type 1 font technology

Type0 + CIDFontType2 (PDF 1.2) - glyph shapes using TrueType/OpenType

TrueType - Embedded TrueType font

Type0 + CIDFontType0 (PDF 1.2) - glyph shapes using Type 1 font technology

MMType1 - Type 1 Multiple Master Fonts
Type3 - glyphs drawn using PDF commands

TrueType Fonts (TTF)
--------------------

https://developer.apple.com/fonts/TrueType-Reference-Manual/
https://en.wikipedia.org/wiki/TrueType
https://github.com/golang/freetype/
https://tex.stackexchange.com/questions/52819/how-do-i-use-truetype-fonts-with-pdftex-using-otftotfm

TrueType is a font standard developed in the late 1980s, by Apple and
Microsoft. TrueType is the most common font format for both the Mac OS
and Microsoft Windows operating systems.

A competitor to Adobe's Type 1 fonts used in PostScript.



OpenType Fonts (OTF)
--------------------

OpenType is a format for scalable computer fonts.  It was built on
TrueType, and is a registered trademark of Microsoft.  OpenType fonts
are used commonly today on the major computer platforms.
